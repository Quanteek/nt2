.. _functors:

Generic Functor
===============

.. rubric:: Description

The following module enables the definition and implementation of tag-dispatched
polymorphic function objects. Such functors are used all through |nt2| as a
simple way to apply a given function to a given type while providing way to
users to extend those behaviors.

.. rubric:: Header File

.. code-block:: cpp

  #include <nt2/include/functor.hpp>
